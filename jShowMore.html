Show More, and animate it

jQuery methods used in this code:

  • .addClass()
  • .animate({}, [speed])
  • .removeClass()

There are two distinct methods used below for expanding a paragraph. It is worth noting that the pages will need to be refreshed in order for a second method to operate correctly after a first method has been implemented.

First, the method used is '.add/removeClass()'. I've demonstrated this method before, however this time we are using it along with the height of the paragraph. Adding a class changes the height from a set amount, in this case '4em', to 'auto'. I have also set the overflow to be hidden. Clicking the link changes the height.

Second, the method used is '.animate({[css property]}, [speed])'.

The CSS property being edited is the height, so effectively the same as the first method, just instead of being one height then another, there is a transitional period where the height changes visibly over time. I have chosen to define the length of time it takes, but if left blank, there would be a default time.
