Adding & Removing a Class

jQuery methods used in this code:

  • addClass.()
  • .removeClass()

Add a Class

The first button adds a class using '.addClass()' (the class used already exists in my CSS code, for the sake of convenience). Note the effect on the text. Next, the second button adds another class. Again, note the effect this has on the targeted paragraph - adding a class preserves any existing classes already assigned to the targeted element, so the first styling is not removed or replaced, just further enhanced. This is the case for the third button also.

Normal Text

Remove a Class

'.removeClass()' does the opposite. It has to be specified which class is to be removed, and also removing one class leaves the other classes intact. In order to remove multiple classes, two classes are seperated by a space in the parentheses and quotation marks. This button demonstrates how one of the classes remains.
