This uses math to establish how much a total cost to a customer would be, factoring in different shipping brackets, and sales tax. The user inputs the shelf price, and this is first converted to a number as opposed to a string that happens to be a number, using 'Number' - perhaps unecessary, but conventional.

Then the function distinguishes which category the cost falls in to, and based on that, the shipping cost is calculated.

Next, the sales tax is added. Since sales tax doesn't count towards qualifying for cheaper shipping, the sales tax is added to the subtotal AFTER shipping is calculated.

Finally, the result is rounded to two decimal places (as currency is), using '.toFixed'. The result is displayed in a paragraph below, using <span> tags.

  • Shipping is free after $20
  • Under $10, the shipping is $1
  • Between $10-20, the shipping is $1 + 5% of the price
  • Sales tax is 10.1%
