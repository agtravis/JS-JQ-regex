Stop an animation

The first button below will start an animation. The second button will stop it dead in its tracks. This will stop the current animation only. If there were more animations coded to take place subsequently, these would continue at the '.stop()' method trigger. See below for multiple animations.

Stop multiple animations

The first button will start a series of 4 animations. Since the page has scrolled down, I have also used jQuery to adjust the starting position so it is visible. The next button is the same '.stop()' code as before, and you'll notice how it only stops the animation that is currently taking place, all the other animations will take place and start from the position of the div on the click of the button. It can be used each time an animation starts on the div.

The third button will stop all animations. The syntax is '.stop(true)'. The last button, '.stop(true, true)', will instantly finish the current animation, and prevent any further animations from happening.
