First, this function stores the div in to a variable. Then, using a 'for' loop, it checks each node in turn to see if it is an element node, using 'nodeType === 1'. Once the counter has hit the appropriate number, the code replaces the original 'innerHTML' with new text, and breaks out of the loop. I have set the counter number target for simplicity.

Since this loop only counts nodes that are elements, it is easier to identify and accounts for whitespace text nodes. However, counting elements is not ideal, as we will see further on, and <p> tags are counted the same as <h#> tags in the element counter.

Instead of an alert, this time the targeted node has its '.innerHTML' altered slightly.
