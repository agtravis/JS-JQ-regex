More ways to select

Some more advanced ways to select elements. These are all selectors involving the DOM.

  • $("p:first") - every paragraph that is a first child of an element
  • $("p:last") - every parapgraph that is a last child of an element
  • $("p:first-of-type") - every paragraph that is the first paragraph in a parent element, regardless of if it is a first child or not
  • $("p:last-of-type") - every paragraph that is the last paragraph in a parent element, regardless of if it is the last child or not
  • $("p:nth-child(#)") - every paragraphs that is the nth child of a parent, where # is a number representing 'n'
  • $("p:nth-of-type(#)") - every paragraph that is the nth of its type within a parent, where # is a number representing 'n'
  • $("p").parent() - every parent of all paragrapgs
  • $("p").siblings() - every element that shares the same parent, except the selected item
