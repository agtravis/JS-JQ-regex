Building a Table From Scratch with javaScript

Keywords used in this code:

  • .appendChild
  • .createElement
  • .createTextNode
  • document
  • . (dot) notation ([object].[property])
  • for loops nested
  • function
  • getElementById
  • if
  • .innerHTML
  • objects concept (var [object name] = {[property]: "value", [property]: #};
  • on/click
  • prompt
  • .setAttribute
  • .toString
  • var

This code not only fills a blank table with content, but also creates the table as well.

First, I set up a div with an ID, and establish the div as a variable.

Next, using 'createElement' I create a 'table' (as a variable), and a 'tbody' (variable) elements.

Creating the cells involves nesting 'for' loops. The parent loop creates the <tr> elements and uses the limit counter to set the amount of rows.

The child loop limit counter sets how many columns, specified as <td>. This loop also specifies what content appears in the cells. I have kept this code, even though at a later stage it is written over and never seen, because it is a useful indicator of what cell appears where, using the variable created for the loops. I also use the variables to define the individual cell ID's, converting the numbers to strings and then using 'setAttribute'. The cell then uses 'appendChild' to fill the cell with the text created above, and then the row uses 'appendChild' to add the cell.

Outside the child loop, the row is added to the table body using 'appendChild', and after the parent loop the table body is added to the table.

Finally I have coded for each cell (designated by the automatically generated ID) replacement text that refers to the objects created in the lines of code preceding the creation of the table. These are simple literal objects with properties to match the amount of cells created. Some content is generated by prompts, simply to have some user input demonstrated.
