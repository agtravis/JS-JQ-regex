Toggle Class (with HTML & CSS)

jQuery methods used in this code:

  • .toggleClass()

This code is a useful way to create a button that will perform alternate tasks. Previously when I have wanted to show and hide something, I have used one button for each. However, by defining a class in CSS, and using '.toggleClass("[classname]")' to alternate between the class being assigned and not assigned, I can make any style features I was be applied and removed.

Below, I have used the code to make images appear and disappear. The class in CSS exclusively is associated to the element's 'display: none'. I have applied the class already to two images, and not to the other two. On clicking the button, it will apply or remove the class as relevant, and therefore it will appear that the images are alternating - they are not, that would involve changing the attribute value of "src", and we are changing the class.

As a bonus I have also coded some javaScript along with jQuery in order to change the text in the button to reflect what the button will do on each click.
