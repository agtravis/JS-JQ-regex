Chaining

Chaining is a shorter way of writing jQuery code that enables multiple methods to happen either consecutively or at the same time, depending on the code. For instance, animations would be successive, but stylistic changes would appear (to the user) to happen at the same time.

The first button performs exactly the same code as we have seen before from an animation tab, except instead of specifying the selector each time, the code is chained together.

Without chaining:
$("div#xmen269img").animate({marginLeft: "+=1200px"}, 5000);
$("div#xmen269img").animate({marginTop: "+=400px"}, 3000);
$("div#xmen269img").animate({marginLeft: "-=400px"}, 3000);
$("div#xmen269img").animate({marginTop: "-=200px"}, 2000);

With chaining:
$("div#xmen269img").animate({marginLeft: "+=1200px"}, 5000).animate({marginTop: "+=400px"}, 3000).animate({marginLeft: "-=400px"}, 3000).animate({marginTop: "-=200px"}, 2000);
