Remove vs. Empty

jQuery methods used in this code:

  • .empty()
  • .remove()

Empty

'.empty()' will leave the targeted element intacts but remove all the contents, or children. Clicking on the first button will remove the demonstration paragraph as far as the user is concerned. Clicking the second button fills the empty paragraph with new text.

Demonstration text



Remove

Clicking the button below will appear to the user to have the same effect as the first button. However, if the button below is clicked, and then the second button is clicked to place new text, it won't work. That is because using '.remove()' completely removes the targeted element, and the node and all its children are gone, therefore there is nothing to fill. A new node would have to be created, with the relevant ID or targeting method, and then the fill button could be used again.
