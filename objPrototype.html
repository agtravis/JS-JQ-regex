A prototype is basically a method or a property coded outside of an object that can be added to an object in the same way that a property can be added. The difference is that each object is accessing the same function when called, as opposed to accessing its own function. This means that scenarios can be applied to multiple objects at the same time.

For example, one of my properties is detailed as 'painted'. I have unilaterally applied a prototype to this property to come up as 'true', however I have also coded and overwrite of 'box3.painted' to show how to supercede the default.
