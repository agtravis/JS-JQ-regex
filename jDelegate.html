Start by adding an item to each list. Next, click on the items in the list. Notice how in the fruit list, all items will disappear on the click, with the exception of the newly appended item. In the second list, ALL items disappear, including the newly appended item. The difference is in the code. The first list has code that includes the element in the selector, whereas in the code for the second list, the element has been delegated from the selector to an argument in the method.

Delegating the element means that the code affects not just original HTML elements, but also elements that have been created after the page was loaded (such as when appending).
