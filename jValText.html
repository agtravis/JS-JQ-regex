Working with fields: inputs, values, and text

jQuery methods used in this code:

  • .text()
  • .val()

The code here uses two methods to do two things. First, the user input is stored as a variable using '.val()'. Next the paragraph displays the variable as its new content using '.text([variable])'. Third, the text of the paragraph is stored in a variable using '.text()', and lastly the variable is placed in the secondary field using '.val([variable])'.

It is worth noting that since the two variables store the exact same string, it could be done with one variable being implemented twice for the different code (A = B, A = C), however, for the sake of demonstration, I have stored a new variable based on the implementation of the first variable (A = B, B = C).
