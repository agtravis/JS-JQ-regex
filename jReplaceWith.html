Replace With

jQuery methods used in this code:

  • .replaceWith()

'.replaceWith()' will take whatever has been targeted and completely remove it, tags and all, from the HTML. Whatever is in the parentheses in the method is what will replace that content.

I have coded two functions, the first demonstrates what happens when no HTML is used. The new text being added is not a paragraph, and will have only styling that applies to the parent node of where the element was previously. Also, you can see from this example that the class is removed and the fact that the text is no longer a paragraph is evident from the way the rest of the content behaves around it.

The second function I have included HTML in the content that replaces the original. It could be anything I wanted, but I have kept the tags, the ID, and the class all the same as the original content, so all that changes is the text. You can see the class stays the same because the formatting remains, and you can see the ID remains because if you follow by clicking the first function, it still targets the same paragraph.
