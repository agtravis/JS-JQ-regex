Animating menu tabs

jQuery methods used in this code:

  • .animate({})

The jQuery code here is very straightforward, it is mostly the CSS which lays all the groundwork.

NOTE: In order to have the tabs move with the scroll, I have used 'absolute' instead of 'fixed' positioning. This meant that I had to move the HTML code for the tabs from this div to a div that had its side right on the edge of the browser screen.

Here is the CSS:
div.tab { [the small tab]
width: 100px; height: 80px; background-color: #333333; [sizing/styling the tab]
position: absolute; right: 0; [positioning the tab ('top' comes later)]
cursor: pointer; [to make it clear it's an anchor]
}

div.tab h6 { [text within the tab]
font-family: arial, helvetica, sans-serif; font-size: 40px; color: white; margin: 19px 0 0 20px; float: left;
}

div#tab1 { top: 250px; }, div#tab2 { top: 330px; }, div#tab3 { top: 410px; } [positioning each successive div below eachother, the difference is the same as the height of the tab]

div.full_tab { [the expanded tab]
position: absolute;
width: 300px;
height: 80px;
background-color: #cc3366;
right: -300px; [to position it exactly offscreen before the jQuery animation]
z-index: 1; [to place this tab on top of the smaller one as it slides in]
cursor: pointer; [were this actually a link, this would be unnecessary]
}

div#full_tab1 { top: 250px; }, div#full_tab2 { top: 330px; }, div#full_tab3 { top: 410px; }

div.full_tab h6 {
font-family: arial, helvetica, sans-serif; font-size: 40px; color: white; float: left; margin: 19px 0 0 20px;
}
