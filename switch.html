This code uses a form to input user data. The the input field is coded to change color while being used using in-line javaScript, also the submit function has been disabled using 'javascript:void(0);'.

Upon 'submit', the 'switch' code runs to establish if the user enters one of two pre-coded choices (converting to lower-case in the process). If either of those two 'case's is entered, a variable is assigned a corresponding string. Anything else is assigned a third string.

Next, an 'if' statement checks to see if either of the two acceptable choices has been made. If not, an alert displays the 3rd message, prompting the user to try again.

If the user has entered correctly, a paragraph displays the appropriate string as a paragraph below.
