These three functions highlight effective ways to remove whitespace text nodes when searching the DOM for an element. First, the function simply accounts for the whitespace and includes it in the node count.

Second, the function first creates an empty array, and then fills it with only the nodes that have a 'nodeType' of 1 (meaning the node is an element), thereby discarding the junk artifacts (the whitespace).

The third button triggers a function that does something similar to the second button, but instead of evaulating the 'nodeType', it is looking at the 'nodeName'. Ensuring that it is converted to lower case, this function's new array only stores elements if they are paragraph elements - in the previous function it was all elements being collected.
