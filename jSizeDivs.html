Measuring window height to size divs & scroll

jQuery methods used in this code:

  • .css()
  • .height()
  • .offset()[.top]
  • .outerHeight()
  • .scrollTop()

In the HTML I have coded 5 empty divs all with unique IDs and matching classes. In the CSS I have given each div its own color.

See the jQuery code for a breakdown line by line, but essentially I have defined the height of the divs and set the buttons to scroll to the top of the next div respectively.

The second button (at the top) is coded to perform a full page scroll, and to animate it.
