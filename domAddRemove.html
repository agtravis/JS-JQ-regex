This function shows how to place new content within the div in different locations, and by using different methods. First a new node is created, then new content is created with which to fill the node. Next, using 'insertBefore', the new node is placed into the div. If the new node is to be placed last, then 'appendChild' is used instead. Using prompts and 'if' statements, the user decides the content of, and where to place the new node.

This button runs code that simply removes the last child of the parent node, in this case the div. However, due to all the whitespace text nodes, sometimes it appears that clicking the button does nothing, when in reality it is removing a whitespace node. For the second button, I have coded a way to eliminate that 'wasted' run of code. I have programmed a 'while' loop that states that each time the node being removed is not an element (i.e. 'nodeType !== 1'), the currently assigned variable skips to the 'previousSibling' automatically.
