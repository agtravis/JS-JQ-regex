Hide/Show

The jQuery code here is very straightforward. One button uses '.hide()' and targets a paragraph to make it disappear. It is effectively the same as the CSS code 'display:none', which means that it is removed from the browser and the rest of the content will act as if it is not there. This is as opposed to changing the visibility of an item, which will mean that the content around it is unaffected. The next button uses '.show()' to create the opposite effect and the paragraph appears again.

This text will disappear and appear on click of the buttons below.





There are a few different methods to make content appear or disappear. Here are some of them.

Fade In/Out



The '.fadeOut()/fadeIn()' method allows something to gradually disappear or appear instead of that instantaneously occurring like with hide/show. A speed can also be specified by detailing in the parentheses at the end of the method either a word ("slow" or "fast"), or a number in milliseconds (so 3000 is 3 seconds). In order to have something be invisible to begin with, and fade in, CSS would have to be used to style the image as "display: none" to begin with.





Slide Up/Down



These two methods are '.slideUp()', and '.slideDown()'. The former takes the image and animates it until it disappears. Inside the paretheses is the speed in the same manner as above. The latter brings it back with the opposite effect.





Toggle



Toggle does the same as hide/show, except that it functions as one button that does both functions. What ever the display status of an item is, it will toggle it to the opposite. this method could be used to alternate between a 'before' and 'after' photograph, for example. Here I have coded two images simply side by side as far as the HTML is concerned, but assigned one a class in CSS to make it 'display: none'. In the jQuery I have set them both to toggle on the same click of the button.



Opacity



When dealing with opacity, the image is always there, even if invisible. That means that unlike when the display is set to 'none', the rest of the content remains unaffected whether or not the image is visible. I have coded two different ways to use the opacity here. First, basically, I am using the '.css({})' method. All this does is change (or set) a style within the CSS code. This makes the opacity zero.

Next, I am using '.animate({}, [speed])' in order to define how long I want a transition from the existing style to the new style to take. This creates a fade effect.
