Date Object

Keywords used in this code:

There is an established array of months, January through December. The user is asked for the month, date, and year of their birth. In order for the array index and month to align (January is the 1st month, but the index is '0'), 1 is subtracted from the month input. All prompts have suggested inputs that can be over-ridden by the user. The user inputs are then concatenated in such a way to which the "Date object' has been standardized, and then using '.getTime()' the milliseconds are calculated.

Next, the milliseconds are used to further calculate for how many years, days, hours, minutes, and seconds the user has been alive, and each time 'Math.floor' is used to leave whole numbers only. These are then displayed as content in the table below using 'document.getElementById' to overwrite the 'innerHTML'.

Then finally, the javaScript calculates the age of the user to the second, but quantifies appropriately for common sense and displays this in the final row. There is a feature to repeat this script once every second using 'setTimeout', to give a live update. There may be a discrepency since a year is calculated at 365.2422 days and the user's year of birth does not take into account leap years. There will be a discrepency due to the fact that the code does not take in to account the exact time the user was born.

Also, as a bonus, an alert is coded if the remaining days is at zero to say "Happy Birthday!".
